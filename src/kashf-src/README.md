To run a project on your localhost, use the following commands:

- `sh run.sh main`
- `sh run.sh alefba`
- `sh run.sh harf`
- `sh run.sh about-us`
- `sh run.sh join-us`
- `sh run.sh contact-us`

You don't need to execute `npm build` manually. Just push and let the Github workflow do the reset.
